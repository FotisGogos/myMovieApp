Get list of all movies /movies GET - A JSON object holding all data about all movies:

{
"_id" : ObjectId("IDNUMBER"),
"title": "Inception",
"descripction": "A thief who steals..."
"director": {
"name" : "Christopher Nolan",
"bio" : "British-American film director...",
"birth_year" : "1970",
"death_year" : ""
},
}
Return data about a single movie by title. /movies/title (title of the movie!!) GET The query is case sensitive! A JSON object holding data about a single movie by title.
Return data about all genres. /genres GET - A JSON object holding data about all genres:

{
"name": "Sci-Fi",
"description": "Lorem ipsum..."
}
Return data about a genre by name. /genre/:name GET Specify genre name in URL. The query is case sensitive! A JSON object holding data about a genre by name.
Return data about all directors. /directors GET - A JSON object holding data about all directors:

{
"name": "Gabriele Muccino",
"bio": "Italian film director."
"birth_year": "1967",

}
Return data about a director by name. /directors/:name GET Specify the name of director. The query is case sensitive! A JSON object holding data about a director.
